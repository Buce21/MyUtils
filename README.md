# MyUtils
My Utils 自定义工具类

StirngUtil - 字符串工具类

NumberUtil - 数字工具类

RegExUtil  - 正则表达式工具类



NumberUtil 数字工具类
1.校验字符串能否装换成数字
2.校验字符串能是否能为整型
3.字符串转int类型
4.字符串转long
5.字符串转short
6.字符串转byte
7.字符串转float
8.字符串转double
9.对一组数字进行取和（数字类型包括int,long,short,float,double）
10.对一组数字进行取平均值（数字类型包括int,long,short,float,double）
11.取一组数字中的最小值（数字类型包括int,long,short,float,double,byte）
12.取一组数字中的最大值（数字类型包括int,long,short,float,double,byte）
